# Personal Portfolio & Virtual CV

A lightweight, single-page React portfolio and virtual CV.

---

## 📌 Features

* **Zero-Build React Architecture:** Runs React 18 directly in the browser.
* **Responsive Layout:** Optimized for desktop and mobile screens.
* **Custom Interactive Components:**
  * Animated typed text role switcher in the hero section.
    
* **Design:** Styled using native CSS variables.

---

## 🛠️ Tech Stack

* **HTML5 / CSS3**
* **JavaScript**
* **React ** (via CDN)

---
